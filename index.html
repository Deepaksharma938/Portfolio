☁ Cloud Sync — Sab Devices Par Data Dikhao

Connected nahi — data sirf is device par save ho raha hai
100% Free — Firebase use karta hai (Google ka, free forever, koi card nahi chahiye):
  1. Jaao console.firebase.google.com → Google account se login karo
  2. Add project → koi bhi naam do (e.g. my-portfolio) → Continue → Continue → Create project
  3. Left menu mein Build → Realtime Database → Create Database → koi bhi location → Start in test mode → Enable
  4. Database URL copy karo — kuch aisa dikhega: https://my-portfolio-default-rtdb.firebaseio.com
  5. Wahi URL niche paste karo → Save & Sync Now button dabao → ho gaya ✓

Note: "Test mode" 30 din baad expire hota hai. Uske baad Realtime Database → Rules tab mein jaake .read aur .write dono ko true karo aur Publish karo.

Pehle se data save hai? Abhi cloud par push karo:

Agar aapne cloud connect karne se pehle details add ki thi, toh ye button click karo. Iske baad har device par aapki details dikhni shuru ho jayengi.

Download Updated Portfolio

Ye button aapka saara data (naam, bio, skills, projects, sab kuch) HTML file mein bake karke download karega. Bas us file ko Cloudflare/Netlify par upload karo — phir duniya mein koi bhi, kisi bhi device par aapki details dekhega. Koi token nahi, koi account nahi.

Kaise kaam karta hai:

01Admin mein jaake apna sara data fill karo (naam, bio, skills, projects, sab)
02Neeche wala button dabao → portfolio_updated.html download hogi
03Cloudflare Pages / Netlify par nayi file upload karo (purani replace karo)
04Done ✓ — ab har device par, incognito mein, mobile mein — sab jagah aapki details!
